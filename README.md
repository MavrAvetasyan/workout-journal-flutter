# Workout Journal Flutter

Flutter-версия журнала тренировок с упором на мобильный интерфейс.

Сейчас в проекте есть:
- экран входа с локальной сессией
- тренировки с активной тренировкой и быстрыми действиями
- справочник упражнений
- история замеров
- локальное хранение данных

Локальный запуск:

```bash
flutter pub get
flutter run -d chrome
```

Web-сборка:

```bash
flutter build web
```
